# AutoHotkey 脚本集合

**简体中文** | [English](README.md)

本仓库包含一系列实用的 AutoHotkey v2.0 脚本，用于增强 Windows 操作系统的使用体验。

## 脚本列表

### 1. monitorwiseAltTab.ahk

#### 显示器感知的 Alt-Tab 窗口切换器

这是一个增强的 Alt-Tab 实现，仅在当前显示器内切换窗口，避免在多显示器环境中切换时跨越屏幕。

#### 功能特点

- **显示器感知**：只显示当前鼠标所在显示器的窗口。
- **美观的 GUI**：显示窗口图标和标题列表。
- **毛玻璃效果**：提供 Windows 10/11 风格的亚克力透明效果。
- **圆角设计**：使用现代化圆角窗口。
- **智能窗口过滤**：自动过滤工具窗口、隐藏窗口和系统窗口。
- **支持最小化窗口**：能够根据恢复位置识别最小化窗口。
- **支持点击激活**：可直接点击弹窗列表中的窗口进行切换。
- **优先选中上一个窗口**：打开时默认选中第二项，更接近系统 Alt-Tab 的使用习惯。

#### 使用方法

- 按下 `Alt + Tab` 打开窗口切换器。
- 保持 `Alt` 键按下，按 `Tab` 键在窗口间循环选择。
- 释放 `Alt` 键切换到选中的窗口。
- 或直接点击列表中的窗口标题。
- 如果当前显示器没有可用窗口，脚本会短暂提示而不是弹出列表。

#### 技术细节

- 使用 `GetWindowPlacement` API 获取窗口位置信息。
- 通过 DWM API 实现毛玻璃和模糊效果。
- 支持多显示器坐标计算。
- 自动清理 GUI 和图标资源。

### 2. mouseButton.ahk

#### 鼠标侧键增强脚本

将鼠标侧键（前进键和后退键）与滚轮及若干键盘映射组合，实现更多快捷功能。

#### 功能映射

| 组合键 | 功能 |
| ------ | ---- |
| **前进键（XButton2）** | 按下时发送 `Shift + NumLock`；松开且未使用滚轮时触发原前进功能 |
| **前进键 + 滚轮** | 横向滚动，是否生效取决于应用程序支持 |
| **后退键（XButton1）** | 按下时不执行操作；松开且未使用滚轮时触发原后退功能 |
| **后退键 + 滚轮上** | `Shift + F3` |
| **后退键 + 滚轮下** | `F3` |
| **Alt + 后退键** | `F12` |
| **Shift + Backspace** | 屏蔽 `Backspace`，只保留 `Shift` |
| **滚轮上下** | 全局滚轮加速，快速连续滚动时会放大滚动次数 |
| **Trae CN 中键** | 在 `Trae CN.exe` 激活时发送 `Ctrl + Alt + B` |

原始前进和后退导航会在松开侧键且未使用滚轮时触发。

#### 设计目的

- 充分利用鼠标侧键，减少键盘操作。
- 防止单独按下 Shift 键时意外触发输入法中英文切换。
- 为常用操作提供快捷方式，例如 IDE 中的查找上一个和查找下一个。
- 增加少量面向工作流的专用映射，同时尽量减少对其他场景的影响。

#### 注意事项

- 需要鼠标具备前进键和后退键。
- 横向滚动功能取决于应用程序是否支持水平滚轮输入。
- 前进键配合滚轮的横向滚动依赖脚本在按住侧键时维持的 `Shift` 状态。
- Trae 专用的中键映射仅在当前活动窗口为 `Trae CN.exe` 时生效。

### 3. rightClick.ahk

#### 快速右键点击脚本

将 `Ctrl + Space` 映射为右键点击。

#### 使用场景

- 快速打开上下文菜单。
- 在触摸板或特殊键盘上替代鼠标右键。
- 提高操作效率。

## 安装与使用

### 系统要求

- Windows 10 或更高版本。
- AutoHotkey v2.0 或更高版本。

### 安装步骤

1. 从 [autohotkey.com](https://www.autohotkey.com/) 安装 AutoHotkey v2.0。
2. 将本仓库克隆或下载到本地目录。
3. 运行任意脚本。
   - 双击 `.ahk` 文件运行。
   - 或右键选择 `Run Script`。
4. 如需开机启动，可将脚本加入启动项。
   - 创建脚本快捷方式。
   - 将快捷方式放入 `%AppData%\Microsoft\Windows\Start Menu\Programs\Startup`。

### 自定义配置

- 每个脚本都可以用文本编辑器打开并修改热键。
- 修改后保存并重新运行脚本即可生效。
- 如果只需要其中某一项功能，也可以只单独运行对应脚本。

## 故障排除

### 脚本不运行

- 确认已安装 AutoHotkey v2.0，可通过 `AutoHotkey.exe /v` 检查版本。
- 检查脚本第一行是否包含 `#Requires AutoHotkey v2.0`。
- 尝试以管理员身份运行脚本。

### 功能无效

- 检查热键是否与其他应用程序冲突。
- 确认鼠标侧键是否正常工作。
- 多显示器环境中，确认显示器设置正确。
- 如果横向滚动无效，请确认目标应用支持水平滚轮输入。
- 如果 `Alt + Tab` 无法持续循环，请确认按 `Tab` 时仍在持续按住 `Alt`。

### 性能问题

- 如果 `monitorwiseAltTab.ahk` 响应较慢，可能是当前打开的窗口过多。
- 确保系统有足够内存。
